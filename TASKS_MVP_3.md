## 30. 3차 MVP 완료 조건

아래 조건을 만족하면 3차 MVP 완료로 판단한다.

- [ ] 앱이 아니라 화면 위 생명체처럼 느껴진다
- [ ] 창의 존재감이 거의 느껴지지 않는다
- [ ] 행동 리듬이 기계적이지 않다
- [ ] idle 상태만 봐도 생동감이 있다
- [ ] 사용자를 의식하는 느낌이 있다
- [ ] 장시간 켜놔도 산만하지 않다
- [ ] Desktop Goose류의 오버레이 감성이 느껴진다
- [ ] UI보다 고양이 존재감이 우선된다
